template<>
struct std::hash< tcb::sudoku::grid >

Standard hash specialisation for grids.

Allows a grid to be placed in a std::unordered_set, or used as the key in a std::unordered_map.

◆ parse() [1/2]

static auto tcb::sudoku::grid::parse ( string_view  str) -> optional< grid >
static

Parse a string to create a new grid.

All characters other than [0-9] and . are ignored. A 0 is interpreted as a ., that is, an unknown value.

Will fail (returning nullopt) if fewer than 81 valid characters could be read.

◆ parse() [2/2]

static auto tcb::sudoku::grid::parse ( std::istream &  istream) -> optional< grid >
static

Parse a stream to create a new grid.

All characters other than [0-9] and . are ignored. A 0 is interpreted as a ., that is, an unknown value.

Will fail (returning nullopt) if fewer than 81 valid characters could be read (i.e. if the end-of-stream was reached).

bool operator< ( const grid &  lhs,
const grid &  rhs 
)
related

Returns true if a grid is "less than" another.

One grid is considered "less than" another if it has more unknown cells. If two grids have the same number of unknown cells then the cells are compared lexicographically.
