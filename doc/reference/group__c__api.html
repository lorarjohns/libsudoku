◆ sudoku_grid_parse()

SudokuGrid* sudoku_grid_parse ( const char *  str)

Parse a null-terminated string to create a new grid.

All characters other than [0-9] and . are ignored. A 0 is interpreted as a ., that is, an unknown value.

Will fail (returning NULL) if fewer than 81 valid characters could be read.

You must free the returned grid with sudoku_grid_free().

SudokuGrid* sudoku_grid_scan ( void  )

Parse a grid from stdin.

All characters other than [0-9] and . are ignored. A 0 is interpreted as a ., that is, an unknown value.

Will fail (returning NULL) if fewer than 81 valid characters could be read before encountering the end of the stream.

You must free the returned grid with sudoku_grid_free().

◆ sudoku_grid_fscan()

SudokuGrid* sudoku_grid_fscan ( FILE *  file)

Parse a grid from a file stream.

All characters other than [0-9] and . are ignored. A 0 is interpreted as a ., that is, an unknown value.

Will fail (returning NULL) if fewer than 81 valid characters could be read before encountering the end of the stream or an IO error.

You must free the returned grid with sudoku_grid_free().

SudokuGrid* sudoku_solve ( const SudokuGrid *  grid)

Attempts to solve the given grid.

If the solve algorithm fails or the supplied grid contains no solutions, returns NULL. Otherwise returns the new, completed grid, which must be freed with sudoku_grid_free()
